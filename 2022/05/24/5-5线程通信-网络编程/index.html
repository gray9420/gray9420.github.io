3.网络编程

3-1 网络相关基础知识

(1).网络应用主流结构

C/S - Client / Server 客户端、服务器

|- 需要开发两套软件

|- 更新维护成本高

|- 充分利用客户端软硬件环境 -LOL

|- 网络处理响应速度快

B/S - Browser / Server 浏览器、服务器

|- 只需要开发一套服务端应用

|-更新维护简单

|-网络处理响应速度相较于C/S慢

(2) 网络通信协议

TCP/IP 协议- 网络七层模型中的传输层

​ |-通信规则 – 类似于生活中普通话

TCP/IP有三次握手和四次挥手的流程，保证数据在网络中传输的安全，稳定性.

(3) 网络编程三要素

a.协议 –> TCP ,Http应用层

b.IP 地址

c.端口 port - 3306
