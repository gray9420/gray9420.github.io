    • 可扩展标记语言 eXtensible Markup Language

    • XML 与 HTML区别

      • XML - 标签可以扩展自定义 HTML标签都是预定义的
      • XML - 区分大小写 ,HTML不区分大小写
      • XML- 注重数据存储, HTML注重数据显示

    • XML语法 - well-forms 形式良好的XML文档

      • 首行是XML文档声明
      • 必须有一个根元素
      • 标签必须完结
      • 属性值使用引号,正确嵌套
      • 如果多个转义字符

    • 验证XML文档 –定义 XML元素合法构建模块

      • dtd
      • xsd
